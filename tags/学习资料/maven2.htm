maven2安装

1.介质准备

         从http://maven.apache.org/download.html 下载最新的maven2.0.9,下载后解压maven到一个特定的目录下，如E:\maven2

2.配置 

  •    设置环境变量,右键点击我的电脑，选择属性-->高级-->环境变量

   

    增加M2_HOME 为E:\maven2,在path 中增加%M2_HOME%\bin;

  • 通过命令行，mvn -version 显示

         Maven version: 2.0.9
        Java version: 1.5.0_14
        OS name: "windows xp" version: "5.1" arch: "x86" Family: "windows"

       表示安装正确。

3.通过maven 管理项目

     普通的Java项目，如基础包等：
         mvn archetype:create   -DgroupId=com.yourcompany  -DartifactId=myproject  -DarchetypeArtifactId=maven-archetype-quickstart
    普通的Web项目，如一个Web项目：
         mvn archetype:create  -DgroupId=com.yourcompany -DartifactId=myproject  -DarchetypeArtifactId=maven-archetype-webapp

     如果你的开发工具是eclipse：则进入通过maven创建的项目目录下，通过mvn eclipse:eclipse 生成eclipse的项目文件
